El login de una aplicación es la página dentro de la plataforma donde el usuario se identifica para iniciar sesión, el objetivo de esta página es autenticar las credenciales para permitir que solo quien está autorizado pueda acceder. Por esta razón es importante que algunas rutas estén protegidas, ya que estos filtros funcionan como barreras de seguridad frente ataques, suplantación o robo. Así mismo, para fortalecer la seguridad de la plataforma es necesario que las personas eviten exponer publicamente contraseñas, información personal, tokens, APIs y otros datos que puedan resultar sensibles.
